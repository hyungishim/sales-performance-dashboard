Sales Performance Dashboard (Excel)

Tool: Microsoft Excel  
Data: Public retail sales dataset (xlsx)

Overview
Built an interactive sales performance dashboard in Excel to analyse sales and profitability across time, regions, and products.

Dashboard Features
- KPI cards summarising total sales, total profit, and profit margin
- Monthly sales and profit trend analysis
- Regional profit comparison
- Product-level sales performance

Key Insights
- Sales and profit show similar trends over time, with noticeable seasonal peaks.
- Some regions contribute strongly to sales but generate lower profit margins.
- A small number of products account for a large share of total sales.
